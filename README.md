
# End-to-End Business Analytics Dashboard

## Project Overview
This project demonstrates a complete analytics workflow from raw data to business insights.

## Tools Used
- Python (Pandas, Matplotlib)
- SQL
- Power BI
- Excel

## Business Problem
Companies need to analyze sales data to understand revenue trends, product performance, and regional sales distribution.

## Key KPIs
- Total Revenue
- Total Profit
- Monthly Sales Trend
- Top Products
- City-wise Sales
- Category Performance

## Project Workflow
1. Data Collection
2. Data Cleaning using Python
3. Data Analysis using SQL
4. Visualization using Power BI
